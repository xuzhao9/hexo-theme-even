## Installation
```
$ cd themes/
$ git clone https://github.com/ahonn/hexo-theme-even even
$ cd even/
$ bin/install
```

Then update your blog's `_config.yml`(yourblog/_config.yml_) to use the theme.

```
# Extensions
## Plugins: http://hexo.io/plugins/
## Themes: http://hexo.io/themes/
theme: even
```
